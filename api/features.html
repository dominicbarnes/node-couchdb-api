node-couchdb-api The CouchDB API Wrapper for Node.js

Feature List

  • Clean, discoverable API
  • Follows Node.js callback conventions
  • Built-in caching support (via CouchDB Etags)
  • Streaming Changes Feed (as well as one-off queries)
  • Attachments support
  • User Account API (via _users db)
  • Utilizes mikeal's request module

Upcoming Features

  • More auth methods (cookie, oauth, etc.)
  • HTTP COPY support
