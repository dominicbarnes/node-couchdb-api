Method list
Description Executes a `_list` function on the current view
HTTP GET /db/_design/design-doc/_list/list-name/view-name

Arguments

Name Type Default Comments
list string Use `ddoc/list` format to choose a `_list` function from another design document.
query Optional hash Query string parameters
(See Official Documentation)
options Optional hash Options for the HTTP request. If needed, explicitly pass `query` as well
callback function See this page for more information 
doc.list("list", { format: "html" }, function (err, response) {
   // your code here...
});
